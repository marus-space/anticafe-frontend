# Модули приложения

Каждый модуль может иметь следующую структуру:

```
myModule
├── components/
├── models/
├── stores/
├── utils/
├── index.ts
└── README.md
```

Подробнее о структуре модуля можно узнать в [документации](https://front-dev.gnivc.ru:9443/gnivc/ff-web/src/branch/master/docs/module-structure.md).
